This plug-in is the fastest, simplest, and easiest way to turn your product review blogs into a source of income -without the hassle and frustration of an affiliate back-office. 
Use it on your WordPress blog or website. The plug-in automatically turns all of your links into affiliate links to make you cash from over 3000 affiliate partners.

Now you can make money automatically from your blog. Just blog normally, and any link is turned into a personal affiliate link, earning you money.

== Installation ==
= For an automatic installation through WordPress: =
1. Go to the 'Add New' plugins screen in your WordPress admin area<br>
2. Search for 'Easy Affiliate'<br>
3. Click 'Install Now'<br>
4. Activate the plugin<br>
	
= For a manual installation via FTP: =
1. Upload the Easy-Affiliate folder to the /wp-content/plugins/ directory<br>
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area<br>
3. start blogging<br>
		
= To upload the plugin through WordPress, instead of FTP: =
Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate it.
